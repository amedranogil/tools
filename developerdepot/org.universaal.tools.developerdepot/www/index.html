Welcome to the universAAL Developer Depot

This site will provide you with all the information and resource you need to quickly get started developing Ambient Assisted Living (AAL) applications using the universAAL platform, and to contribute to the development of the platform.

Getting started developing AAL applications!

The universAAL platform provides both a runtime platform, a set of tools for development of applications for this platform, and an online store for publishing your applications in collaboration with service providers.

To get started with the development:

  • Sign an account to get access to all resources
  • Learn the basics about how to use the platform from our online developer handbook
  • Read about and install the available developer tools by having a look at the tools wiki pages or using the tools tutorial
  • Check out the official example application
  • Check out other open source applications for the platform (link coming soon)

Quick links for configuring the tools on your own:

  • Eclipe update site: http://depot.universaal.org/eclipse-update/
  • Maven repository: http://depot.universaal.org/maven-repo/

Learning more about the platform

  • Visit the project web site, at: http://www.universaal.org/
  • Visit the uStore to look at the available applications and to register as a developer, at: http://ustore.universaal.org/
  • Read about the architectural design of the platform
  • Browse the API documentation and project reports
  • Browse the UML model documenting the platform
  • Read about universAAL Reference Use Cases where universAAL Platform features are described technically

Contributing to the development of the platform

The universAAL platform is an open source platform, with development being led by the universAAL project.

  • Register as a developer. Please note that access is controlled at the level of the sub-projects, and at access to sub-projects may currently be limited to universAAL project participants.
  • Browse the sub-projects. The development of the platform is divided in several sub-projects, where each sub-project provides wiki pages, SVN repository, issue trackers, etc.
  • Visit the uStore to look at the available applications and to register as a developer (link coming soon)
  • Access our Maven repository through the Nexus Repository Manager
  • Use the general-purpose tracker to view or register issues that are not related to a specific sub-project
